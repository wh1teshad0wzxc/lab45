# Airport Tariff App

Десктопное приложение для управления тарифами перевозок в аэропорту.

## Функциональность
- отображение списка тарифов;
- добавление тарифов перевозки;
- удаление выбранных тарифов;
- проверка корректности вводимых данных;
- работа в многооконном режиме.

## Используемая база данных
Для хранения данных используется реляционная база данных PostgreSQL.
База данных содержит информацию о тарифах перевозки, включая направление, стоимость, тип груза и максимально допустимый вес.
Связь приложения с базой данных реализована с использованием JDBC.

## Структура проекта
- model — классы тарифов (Tariff);
- ui — графический интерфейс (MainFrame, TariffDialog);
- db — логика работы с данными;
- Main — запуск приложения.

## Запуск
Проект запускается из среды IntelliJ IDEA через класс Main.

## Технологии
- Java
- Swing
- PostgreSQL
